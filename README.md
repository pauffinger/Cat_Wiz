## üìã Overview

Cat_Wiz is a computational toolkit designed for structural biologists working with RNA-metal ion interactions. The software provides advanced stereochemistry-guided methods for identifying, analyzing, and annotating magnesium ions (Mg¬≤‚Å∫) in RNA crystal and cryo-EM structures.

### ‚ú® Key Features

- üéØ **Precise Localization**: Accurately identifies Mg¬≤‚Å∫ ion positions in RNA structures
- üîç **Diagnostic Tools**: Comprehensive analysis of metal coordination geometry
- üìù **Automated Annotation**: Streamlined annotation workflow for structural data
- üß™ **Stereochemistry-Guided**: Uses rigorous geometric and chemical validation
